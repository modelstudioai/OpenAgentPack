# Support

Use the public channels that match your request:

- Ask usage questions and discuss designs in [GitHub Discussions](https://github.com/modelstudioai/OpenAgentPack/discussions).
- Report reproducible bugs with the [bug report form](https://github.com/modelstudioai/OpenAgentPack/issues/new?template=bug-report.yml).
- Propose product changes with the [feature request form](https://github.com/modelstudioai/OpenAgentPack/issues/new?template=feature-request.yml).
- Report vulnerabilities privately by following [SECURITY.md](./SECURITY.md).

Please do not include credentials, private configuration, or unredacted provider responses in public posts.
